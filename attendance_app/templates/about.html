Understanding the Math Behind the Results

Why Understanding the Math Matters

This page explains the mathematical concepts and formulas used to calculate the results displayed on the main page. By understanding the calculations, you can gain deeper insights into how we analyze the data and how the conclusions are derived.

Decreased Percentage Calculation

The decreased percentage is calculated using the formula: 


            Decreased Percentage = ((Previous Value - Current Value) / Previous Value) * 100

This formula calculates how much a value has decreased relative to its previous value. The result is expressed as a percentage to make it easy to interpret the change.

For example, if the previous value was 80 and the current value is 60, the decreased percentage would be: 


            Decreased Percentage = ((80 - 60) / 80) * 100 = 25%

Decrement Per Day Calculation

The decrement per day is calculated by measuring the difference in values between consecutive days. The formula is: 


            Decrement Per Day = Current Day Value - Previous Day Value

This formula gives us the change in value for each day. It helps in tracking the daily fluctuations in the data.

For example, if the value on day 1 is 100 and on day 2 it is 80, the decrement per day would be: 


            Decrement Per Day = 80 - 100 = -20

The Formulas at a Glance

To summarize, here are the formulas used in the analysis:

  • Decreased Percentage: 
    ((Previous Value - Current Value) / Previous Value) * 100
  • Decrement Per Day: 
    Current Day Value - Previous Day Value

Example Calculation

Let's go through an example where we calculate both the decreased percentage and decrement per day:

Example 1:
  • Day 1 Value: 100
  • Day 2 Value: 80
Decreased Percentage:
((100 - 80) / 100) * 100 = 20%
Decrement Per Day:
80 - 100 = -20

Conclusion

Understanding the math behind the results helps you interpret the data more accurately. The decreased percentage gives you an idea of the magnitude of change, while the decrement per day helps you track daily fluctuations. With this knowledge, you can make more informed decisions based on the analysis.
